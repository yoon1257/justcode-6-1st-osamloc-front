## 4. 적용 기술 및 구현 기능

### 4-1. 적용 기술
+ Front-end : JavaScript, React.js, scss 
+ Back-end : JavaScript, Node.js, Express, bcrypt, JWT
+ Database : MySQL 
+ HTTP : Postman
+ Common : Slack, Zep, Notion, Git&Github


### 4-2. 구현 기능

맡은 기능 
   - 로그인
      - 로그인 ui 구현
      - 로그인 api 
          - input창에 조건에 맞는 값이 모두 입력시 로그인 button 색 변화기능 구현
          - 유효성검사 
   
   - 회원가입
      - 약관동의과 관련되어 필수인 부분이 전부 클릭이 되어야 가입버튼 눌러지도록 구현
      - input 창에 유효성 검사를 통해 조건이 맞게 기입이 되면 안내문구의 색상 변화 기능 구현
      
   - Footer
      - Footer UI 구현
      - 로고 디자인
